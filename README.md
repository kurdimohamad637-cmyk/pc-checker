# 🎮 FiveM PC Checker - World's Best PC Health Diagnostic Tool\n\nA professional, real-time PC health check and diagnostics tool optimized specifically for FiveM gaming performance.\n\n## ✨ Features\n\n- **🏆 FiveM Performance Score** - Get a detailed score (0-100) of your PC's FiveM compatibility\n- **⚙️ CPU Monitoring** - Real-time CPU usage, cores, threads, and speed information\n- **🧠 RAM Analysis** - Complete memory statistics including used, free, and total RAM\n- **🎨 GPU Detection** - Identify your graphics card and memory\n- **💾 Storage Analysis** - Monitor disk space and usage across all drives\n- **🌐 Network Diagnostics** - Check internet speed and network interface statistics\n- **📊 Beautiful Dashboard** - Modern, responsive UI with real-time updates\n- **💡 Recommendations** - Get optimization tips based on your system performance\n\n## 🚀 Quick Start\n\n### Prerequisites\n- Node.js (v14 or higher)\n- npm or yarn\n\n### Installation\n\n```bash\n# Clone the repository\ngit clone https://github.com/kurdimohamad637-cmyk/pc-checker.git\ncd pc-checker\n\n# Install dependencies\nnpm install\n\n# Start the server\nnpm start\n```\n\nThe application will be available at `http://localhost:3000`\n\n### Development\n\n```bash\n# Run with auto-restart on file changes\nnpm run dev\n```\n\n## 📋 System Requirements\n\n### Minimum for FiveM\n- **OS**: Windows 10/11 or Linux\n- **CPU**: 4 cores @ 2.5GHz+\n- **RAM**: 8GB (16GB recommended)\n- **GPU**: Dedicated graphics card (VRAM: 2GB+)\n- **Disk**: 100GB free space\n- **Network**: 5+ Mbps stable connection\n\n## 🎮 FiveM Optimization Tips\n\nBased on your PC Checker score, here are recommended optimizations:\n\n### High CPU Usage\n- Close background applications (Discord, Chrome, etc.)\n- Disable unnecessary startup programs\n- Update GPU drivers\n- Reduce in-game graphic settings\n\n### High RAM Usage\n- Increase available RAM or reduce applications\n- Clear system cache\n- Disable resource-heavy programs\n- Restart your PC if usage exceeds 85%\n\n### Poor Network Performance\n- Use wired Ethernet connection\n- Check internet speed (test at speedtest.net)\n- Move closer to router\n- Close bandwidth-heavy applications\n\n## 🛠️ API Endpoints\n\n### GET `/api/diagnostics`\n\nReturns complete system diagnostics.\n\n**Response:**\n```json\n{\n  \"timestamp\": \"2024-01-10T12:00:00Z\",\n  \"system\": {...},\n  \"cpu\": {...},\n  \"memory\": {...},\n  \"gpu\": {...},\n  \"disk\": [...],\n  \"network\": [...],\n  \"fivem\": {\n    \"score\": 95,\n    \"status\": \"EXCELLENT\",\n    \"recommendations\": [...]\n  }\n}\n```\n\n## 📦 Technologies Used\n\n- **Backend**: Node.js + Express.js\n- **System Info**: systeminformation library\n- **Frontend**: Vanilla JavaScript + CSS3\n- **Design**: Modern gradient UI with animations\n\n## 🤝 Contributing\n\nContributions are welcome!
